# Verboo 词库数据库结构

## 数据库文件
`verboo_vocabulary.db` (SQLite3)

## 表结构

### words 表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER | 主键，自增 |
| word | TEXT | 单词（唯一，小写） |
| phonetic | TEXT | 音标 |
| definition_en | TEXT | 英文释义 |
| definition_cn | TEXT | 中文释义 |
| pos | TEXT | 词性，统一格式，多个用逗号分隔（如 `n,v,adj`） |

#### 难度等级

| 字段名 | 类型 | 说明 |
|--------|------|------|
| cefr_level | TEXT | CEFR 等级：A1/A2/B1/B2/C1/C2 |
| collins_star | INTEGER | 柯林斯星级：0-5，数字越大越常用 |
| coca_rank | INTEGER | COCA 词频排名，数字越小越常用 |
| bnc_rank | INTEGER | BNC 词频排名，数字越小越常用 |

#### 考试标签（0/1 布尔值）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| is_oxford_3000 | INTEGER | 是否属于牛津 3000 核心词 |
| is_cet4 | INTEGER | 是否属于 CET-4 词汇 |
| is_cet6 | INTEGER | 是否属于 CET-6 词汇 |
| is_zk | INTEGER | 是否属于中考词汇 |
| is_gk | INTEGER | 是否属于高考词汇 |
| is_ky | INTEGER | 是否属于考研词汇 |
| is_toefl | INTEGER | 是否属于托福词汇 |
| is_ielts | INTEGER | 是否属于雅思词汇 |
| is_gre | INTEGER | 是否属于 GRE 词汇 |

#### 扩展信息

| 字段名 | 类型 | 说明 |
|--------|------|------|
| exchange | TEXT | 词形变化（时态、复数等） |
| data_sources | TEXT | 数据来源标记（如 `ecdict,coca,cefr`） |

## 索引

- `idx_word` - word 字段索引，加速单词查询
- `idx_cefr` - cefr_level 字段索引
- `idx_coca` - coca_rank 字段索引
- `idx_oxford` - is_oxford_3000 字段索引

## 词性标识说明

| 标识 | 含义 |
|------|------|
| n | 名词 (noun) |
| v | 动词 (verb) |
| adj | 形容词 (adjective) |
| adv | 副词 (adverb) |
| prep | 介词 (preposition) |
| pron | 代词 (pronoun) |
| conj | 连词 (conjunction) |
| det | 限定词 (determiner) |
| interj | 感叹词 (interjection) |
| art | 冠词 (article) |

## CEFR 等级说明

| 等级 | 含义 | 大致对应 |
|------|------|----------|
| A1 | 入门级 | 小学/初中 |
| A2 | 基础级 | 初中 |
| B1 | 中级 | 高中/CET-4 |
| B2 | 中高级 | CET-6/雅思5.5-6.5 |
| C1 | 高级 | 专八/雅思7-8 |
| C2 | 精通级 | 母语水平 |

## 数据源

| 来源 | 说明 |
|------|------|
| ecdict | ECDICT 开源词典，提供基础词汇数据、音标、释义、考试标签 |
| coca | COCA 词频表，美式英语词频排名 |
| cefr | Words-CEFR-Dataset，CEFR 等级数据 |
| wordnet | NLTK WordNet，补充缺失的词性数据 |

## 示例查询

```sql
-- 查询高考词汇
SELECT word, definition_cn, cefr_level FROM words WHERE is_gk = 1;

-- 查询 B1 级别的词汇
SELECT word, definition_cn FROM words WHERE cefr_level = 'B1';

-- 查询 COCA 前 1000 高频词
SELECT word, coca_rank, definition_cn FROM words
WHERE coca_rank > 0 AND coca_rank <= 1000
ORDER BY coca_rank;

-- 查询同时是托福和雅思的词汇
SELECT word, definition_cn FROM words WHERE is_toefl = 1 AND is_ielts = 1;
```
