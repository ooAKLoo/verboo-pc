# Verboo 产品介绍文档

## 一、产品概述

**产品名称**：Verboo

**一句话定位**：面向英语学习者的智能视频学习助手

**产品形态**：跨平台桌面应用（macOS / Windows / Linux）

**核心理念**：将碎片化的视频观看时间转化为高效的英语学习时刻

---

## 二、目标用户

| 用户群体 | 核心需求 | 使用场景 |
|---------|---------|---------|
| 考研学生 | 学术词汇、逻辑连接词 | 观看英语讲座、学术视频 |
| 职场人士 | 商务表达、会议常用短语 | 观看商业演讲、行业视频 |
| 影视爱好者 | 口语化表达、文化梗 | 观看美剧、YouTube视频 |
| 英语教师 | 教学素材收集、字幕截图 | 制作教学课件 |

## 三、核心功能

### 1. 内置智能浏览器
- 集成视频浏览器，无需切换应用
- 支持 YouTube、B站等主流平台
- 自动追踪视频播放进度
- 最近访问记录快速回溯

### 2. 字幕管理系统
- **多源字幕获取**：YouTube 自动字幕、B站字幕、本地 SRT/VTT 导入
- **AI 字幕识别**：B站视频智能语音转字幕
- **字幕时间轴同步**：点击字幕即可跳转到对应视频时刻
- **字幕库持久化**：按视频存储，随时调用

### 3. 智能截图功能
- **一键截图**：快捷键 `S` 快速截取当前画面
- **字幕自动叠加**：智能匹配当前时间点字幕
- **学习标记**：
  - `I` 键 → 标记为「重点」
  - `D` 键 → 标记为「难点」
- **后期编辑**：支持多字幕选择、样式调整、位置微调

### 4. 词汇学习模块
- **智能词汇识别**：自动提取视频中的学习价值词汇
- **四维评估体系**：
  - 语言价值：词本身是否值得学
  - 场景价值：在视频语境中的重要性
  - 表达价值：学后是否更地道
  - 用户价值：对个人的实用性
- **AI 辅助判定**：生成学习理由和适用场景

### 5. 资源库管理
- 统一管理截图、字幕、词汇
- 跨平台内容聚合
- 全文搜索与分类筛选
- 一键导出

## 五、核心卖点

### 1. 无感学习体验
> 在正常观看视频的过程中，无需打断，一键完成截图、标记、词汇收集

### 2. 智能而非机械
> 不是简单的词频统计，而是 AI 驱动的多维度学习价值评估

### 3. 数据本地优先
> 所有学习数据存储在本地，保护用户隐私，无网络也能复习

### 4. 跨平台聚合
> YouTube、B站等多平台内容统一管理，告别碎片化

### 5. 学习闭环
> 截图 → 字幕 → 词汇 → 复习，形成完整的学习资产沉淀

## 六、差异化优势

| 对比维度 | 传统方式 | Verboo |
|---------|---------|--------|
| 截图 | 手动截图 + 手动添加字幕 | 一键截图 + 自动字幕叠加 |
| 字幕 | 第三方网站下载 | 内置获取，自动关联视频 |
| 词汇 | 手动查词 + 笔记 | 智能识别 + 价值评估 |
| 管理 | 分散在各个应用 | 统一资源库 |
| 复习 | 无系统化方案 | 标记系统 + 资源关联 |

## 七、用户旅程

```
启动应用
    ↓
输入视频 URL / 选择最近观看
    ↓
浏览器加载视频 + 自动获取字幕
    ↓
观看过程中：
  • 遇到精彩画面 → 按 S 截图
  • 遇到重点知识 → 按 I 标记重点
  • 遇到生僻表达 → 按 D 标记难点
    ↓
查看学习面板 → 浏览 AI 推荐词汇
    ↓
进入资源库 → 复习截图和词汇
    ↓
导出学习资料 → 形成个人知识库
```

## 十一、产品价值主张

### For 用户
> Verboo 是一款让你在看视频时自然积累英语知识的智能助手。无需改变观看习惯，只需一个快捷键，精彩瞬间连同字幕一起保存，AI 自动识别值得学习的词汇，所有学习资料统一管理，随时复习。

### For 市场
> 在英语学习工具赛道，Verboo 开创了「视频原生学习」的新范式。不是让用户去学习 App 里看视频，而是在用户看视频的地方嵌入学习能力。通过 AI 驱动的智能识别和本地化的数据管理，为用户提供真正无感、高效的英语学习体验。
